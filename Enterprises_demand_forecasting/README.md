# Enterprise Demand Forecasting System

## 🚀 Overview
An end-to-end Machine Learning solution designed to predict enterprise product demand. This project covers the full pipeline from data preprocessing and feature engineering to model deployment using Flask.

## 🛠 Tech Stack
* **Language:** Python
* **ML Framework:** Scikit-Learn
* **Deployment:** Flask
* **Environment:** Git, Jupyter Notebooks

## 📂 Project Structure
- `notebooks/`: Exploratory Data Analysis and Model Training pipelines.
- `models/`: Serialized ML models (Note: Large files excluded via .gitignore).
- `templates/` & `static/`: Frontend interface for the forecasting application.
- `app.py`: Flask backend to serve predictions.

## 📊 Key Features
* Automated data cleaning pipeline to handle missing values and inconsistencies.
* Feature engineering focused on temporal trends and historical demand.
* Scalable model architecture for enterprise-level forecasting.

## 🚀 How to Run
1. Clone the repository: `git clone <your-repo-url>`
2. Install dependencies: `pip install -r requirements.txt`
3. Launch the app: `python app.py`
